> 收集并整理各种 AI 编程工具和 AI 模型的系统提示词（System Prompts），帮助开发者深入理解 AI 工具的工作原理，提升 prompt 工程技能。

## 📚 内容概览

本仓库包含 **30,000+ 行** AI 工具系统提示词，涵盖多种主流 AI 编程助手和模型。

## 📁 项目结构

```
The-Art-Of-Prompt-ZH/
├── LICENSE                          # MIT 许可证
├── README.md                        # 项目说明文档
├── Claude Code(Piebald-AI)/         # Claude Code 系统提示词

    ├── Windsurf/                    # Windsurf
    ├── VSCode Agent/               # VSCode Agent

```

## 🚀 快速开始

### 克隆仓库

```bash
git clone https://github.com/qq626928256/The-Art-Of-Prompt-ZH.git
cd The-Art-Of-Prompt-ZH
```

### 浏览提示词

```bash
# 查看 Claude Code 提示词
ls "Claude Code(Piebald-AI)/system-prompts/"

# 查看其他 AI 工具提示词
ls system-prompts-and-models-of-ai-tools/
```

## 📖 使用说明

### 对于开发者

1. **学习参考** - 阅读各 AI 工具的系统提示词，理解其工作原理
2. **Prompt 工程** - 借鉴优秀提示词设计，提升自己的 prompt 技能
3. **二次开发** - 基于现有提示词进行定制和优化

### 对于 AI 研究者

1. **分析对比** - 对比不同 AI 工具的提示词策略
2. **趋势分析** - 了解 AI 编程助手的演进方向
