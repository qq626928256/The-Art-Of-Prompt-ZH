使用鼠标和键盘与 Web 浏览器交互，并截图。如果你没有有效的标签页 ID，请先使用 tabs_context_mcp 获取可用的标签页。
* 每当你打算单击元素（如图标）时，你应该参考截图以在移动光标之前确定元素的坐标。
* 如果你尝试单击程序或链接但加载失败，即使在等待之后，也尝试调整单击位置，使光标的尖端视觉上落在你想要单击的元素上。
* 确保使用光标尖端在元素中心单击任何按钮、链接、图标等。除非要求，否则不要单击框的边缘。
